# Anime Score Scraper (Web)
Scrapes anime scores from MyAnimeList, Anime Planet, and AniList using BeautifulSoup.

Demo Link: https://anime-scores.herokuapp.com/

## Tech Stack
- Frontend: HTML, CSS
- Backend: Flask (Python)
- Web Scraping Tool: BeautifulSoup4
- Deployed on Heroku
